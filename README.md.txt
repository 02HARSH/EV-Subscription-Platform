# EV Subscription Platform 🚗⚡

A full-stack microservices-based EV subscription system built during training at TVS Motor Company.

## Tech Stack
- Java, Spring Boot
- React (Vite)
- Microservices architecture
- Razorpay integration

## Services
- Battery Service
- Payment Service
- Email Notification Service

## Run

### Backend
Run each service:
mvn spring-boot:run

### Frontend
cd frontend
npm install
npm run dev
